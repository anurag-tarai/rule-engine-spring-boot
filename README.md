# Rule Engine Spring Boot

A Spring Boot backend project implementing a simple rule engine for evaluating transactions based on rules.

## Tech Stack
- Java 17
- Spring Boot
- Maven

## Project Structure
src/main/java/com/leadicon/ruleengine
├── controller
├── model
├── service

## How to Run
```bash
mvn clean install
mvn spring-boot:run
```
